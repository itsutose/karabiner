# Karabiner-Elements Configuration

My personal Karabiner-Elements settings.

## Usage

### Save current config to dotfiles
```bash
karabiner-save
```

### Apply dotfiles config to system
```bash
karabiner-apply
```

## Files

- `karabiner.json` - Main configuration file
- `sync.sh` - Sync script for managing config
- `Default_profile_karabiner.json` - Backup of default profile
- `dev_profile_karabiner.json` - Backup of dev profile

## Setup

1. Clone this repository to `~/dotfiles/karabiner-element`
2. Run `karabiner-apply` to apply settings
3. Restart Karabiner-Elements
